## 🤖 Agent Details

### Node 1: AISHA (Intake Agent)
**Purpose:** Extract structured data from raw user messages

**Input:** Raw text message from WhatsApp/web
**Output:** Structured JSON with extracted variables

**Extracted Fields:**
- `tenant_name` - Full name of tenant
- `landlord_name` - Full name of landlord
- `rent_amount` - Monthly rent with currency
- `property_address` - Complete property address
- `agreement_start_date` - Start date (YYYY-MM-DD)
- `agreement_duration` - Duration (e.g., "11 months")
- `security_deposit` - Deposit amount with currency

**Prompt Strategy:**
- Strict JSON output enforcement
- Handles Indic names and addresses
- Date standardization
- Null values for missing fields

---

### Node 2: DRAFTER (Legal Architect)
**Purpose:** Generate complete rental agreement document

**Process:**
1. Receives extracted JSON from Aisha
2. Performs vector similarity search in PostgreSQL
3. Retrieves best-matching legal template
4. Injects variables into template using LLM
5. Generates formatted markdown document
6. Converts to PDF using ReportLab

**Features:**
- RAG-powered template selection
- Maintains legal language and formatting
- Preserves all clauses from templates
- Adds execution date automatically
- Notes missing information if any

**Output:**
- Markdown file (.md)
- PDF document (.pdf) with signatures section

---

### Node 3: AUDITOR (Quality Assurance)
**Purpose:** Verify drafted agreement accuracy

**Audit Checks:**
✅ Tenant name appears correctly
✅ Landlord name appears correctly  
✅ Rent amount stated accurately
✅ Property address complete
✅ Agreement dates specified
✅ Security deposit mentioned
✅ Legal consistency

**Scoring:**
- Score ≥ 85 + no critical issues = **PASS**
- Score < 85 or critical issues = **FAIL**

**Conditional Routing:**
- **PASS** → Finish workflow
- **FAIL** → Loop back to Drafter (max 3 revisions)

## 🔧 Technical Implementation

### State Schema (AgentState)

```python
class AgentState(TypedDict):
    # Input
    raw_input: str
    sender: str
    
    # Extracted by Aisha
    tenant_name: Optional[str]
    landlord_name: Optional[str]
    rent_amount: Optional[str]
    property_address: Optional[str]
    agreement_start_date: Optional[str]
    agreement_duration: Optional[str]
    security_deposit: Optional[str]
    extracted_json: Optional[Dict[str, Any]]
    
    # Generated by Drafter
    template_id: Optional[int]
    template_content: Optional[str]
    drafted_document: Optional[str]
    pdf_path: Optional[str]
    
    # Audit results
    audit_passed: bool
    audit_feedback: Optional[str]
    revision_count: int
    
    # Metadata
    current_node: str
    errors: List[str]
    timestamps: Dict[str, str]
```

### Graph Construction

```python
workflow = StateGraph(AgentState)

# Add nodes
workflow.add_node("aisha_intake", aisha_intake_node)
workflow.add_node("drafter", drafter_node)
workflow.add_node("auditor", auditor_node)

# Set flow
workflow.set_entry_point("aisha_intake")
workflow.add_edge("aisha_intake", "drafter")
workflow.add_edge("drafter", "auditor")

# Conditional routing
workflow.add_conditional_edges(
    "auditor",
    should_revise,
    {"finish": END, "revise": "drafter"}
)

app = workflow.compile()
```

## 🚀 Usage

### Via FastAPI Endpoint

```bash
# Start the backend server
cd /workspace/ag-associates-ai/backend
uvicorn main:app --reload --host 0.0.0.0 --port 8001
```

**Generate Agreement:**
```bash
curl -X POST http://localhost:8001/api/generate-agreement \
  -H "Content-Type: application/json" \
  -d '{
    "raw_input": "I need a rental agreement. Tenant: Rahul Patil, Landlord: Suresh Deshmukh, Property: Flat 201, Pune, Rent: ₹28,000, Duration: 11 months",
    "sender": "user_123"
  }'
```

**WhatsApp Webhook (via n8n):**
```bash
curl -X POST http://localhost:8001/webhook/whatsapp \
  -H "Content-Type: application/json" \
  -d '{
    "message": "Need rental agreement for my Pune property...",
    "sender": "whatsapp_user_456"
  }'
```

### Direct Python Execution

```python
from agents import process_rental_request

result = process_rental_request(
    raw_input="Your rental agreement request here...",
    sender="test_user"
)

if result["success"]:
    print(f"Document: {result['document_path']}")
    print(f"Audit: {'PASSED' if result['audit_passed'] else 'FAILED'}")
```
